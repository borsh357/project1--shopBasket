# Basket component for online shop

## install

```
npm i
```

## Scripts

**Dev-server + watch**

```
npm run dev-mode
```

**build project**

```
npm run build
```

**run unit tests**

```
npm run test
```

**validate js/css/html and build**

```
npm run valid-build
```

**Publish to github pages**

```
npm run deploy
```
